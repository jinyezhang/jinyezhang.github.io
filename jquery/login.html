# 登录

提示

登录功能是 PC 端或手机端经常会用到的功能，登录功能往往与手机验证码的获取有关系，在 60s 倒计时时间内，无法再次获取。倒计时结束后可再次获取验证码……

下面是 js 部分的代码

var time = 60;
//调取获取验证码的接口，如果调取成功，则开始倒计时
...
var timer = setInterval(()=>{
    if(time>0){
        time--;
    }else{
        clearInterval(timer);
    }
},1000)
...
