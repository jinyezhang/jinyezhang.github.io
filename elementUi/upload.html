# 文件上传

提示

在做后台管理系统的时候，经常需要用到文件或者图片的上传，如果使用框架的话，比如 elementUi 框架，则可以通过组件中的 upload 来上传。

还有一种方式就是直接使用input组件，当input组件的type为file时，就可以进行文件或者图片的上传了

# 1.触发选择文件的 html 部分

整个页面都是用的elementUi的框架，因此部分组件是用的elementUI中的组件形式，例如下面的按钮组件:

<el-button size="mini" type="success" @click="importFn" icon="el-icon-upload" :loading="importLoading" class="searchBtn">导入</el-button>
1

分析一下这个组件：

  1. size：组件大小，mini 表示迷你小按钮，小点会觉得好看些
  2. type：success，表示按钮的主题颜色是绿色。
  3. click="importFn" ：当点击此按钮时，会出现 importFn 函数
  4. icon：表示按钮前面有个小图标
  5. loading：表示按钮在使用过程中有个加载中的状态，防止重复点击。
  6. class：这个是一个样式类名，这个无所谓的……

# 2.input 上传文件的 html 部分

<div style="opacity: 0">
  //此时opacity:0;表示隐藏，不展示
  <input
    v-if="fileInput"
    ref="fileUpload"
    accept=".xls,.xlsx,.csv"
    multiple
    style="width: 0px; height: 0px"
    type="file"
    v-on:change="addFile($event)"
  />
</div>

现在来讲解一下参数：

  1. opacity:0;表示这部分是隐藏的，并不在页面中展示
  2. fileInput ：表示 input 的弹窗是通过 fileInput 的值来展示获取隐藏的
  3. ref="fileUpload"：vue 中可以通过 ref 来获取到这个 dom 元素。这个在后面有用处
  4. accept=".xls,.xlsx,.csv" 这个表示可以支持的文件类型，其实这个并没有完全的限制住文件的类型 因为还是可以选择 所有文件来进行选择的，这个需要在上传文件之前针对文件的类型进行后续的判断。
  5. multiple ：表示可以支持多选
  6. style="width: 0px; height: 0px" ：表示组件不展示，大小为 0 即为不展示
  7. type="file" 上传文件
  8. v-on:change="addFile($event)" ，这个 input 组件有两个函数可以监听，一个是 change，另一个是 Input，如果是其他类型的 input 组件，比如 text number 之类的，都是要执行的是 Input 函数，此时是上传文件，所以走的应该是 change 事件。

# 3.js 部分

// 导入函数
 importFn() {
    this.fileInput = true;
    this.$nextTick(() => {
    	//页面渲染完成后，触发点击事件，弹出对话框
        this.$refs.fileUpload.click();
    });
},
//上传文件
addFile(file) {
    console.log(file);
    var files = file.target.files[0];//虽然我的input支持多个上传，但是此时我还是只处理了第一个文件，多选的参数可以去掉了,如果是多个文件的话，就需要在此处进行for循环遍历，依次进行文件的上传
    console.log(files);
    if (!files) {
        return;
    }
    //此处的文件类型，我判断的是 excel表格，如果是图片，则需要进行图片格式的判断
    if (files.type === "application/vnd.ms-excel" || files.type === "application/vnd.openxmlformats-officedocument.spreadsheetml.sheet") { console.log("格式正确"); } else {
        this.$message.error("文件格式错误！");
        return;
    }
    const isLt50M = files.size / 1024 / 1024 < 50;
    if (!isLt50M) {
        this.$message.error("上传文件大小不能超过 50MB!");
        return;
    }
    this.importLoading = true;
    //由于后端接口支持的是上传formData二进制的数据，因此通过 new FormData的方式来实例化，然后将二进制数据绑定到file属性名上面，
    var formData = new FormData();
    formData.append('file', file.target.files[0])
    $.ajax({
        type: 'post',
        url: "/PurchaseOrder/Import", //这个是我们这边后端提供的接口地址，需要更换成对应的
        data: formData,
        cache: false,
        processData: false,
        contentType: false,
    }).success(res => {
        if (res.Success) {
            this.$message.success(res.Message || "导入成功");
            this.importLoading = false;
        } else {
            this.$message.error(res.Message);
            this.importLoading = false;
        }
        this.$refs.fileUpload.value = null; //这个代码是为了解决上传完一个文件后，再次上传此文件无法触发上传的问题
    })
},
